1. Static Position

Static
Static is the default position. The element just stays where it appears in the HTML. You can't move it using top/left/right.

2. Relative Position

Relative
Relative means: stay in your spot, but you can move yourself a bit from there. This blue box has moved 20px down and 20px to the right.

3. Absolute Position

Absolute
Absolute takes the element out of the normal layout and places it exactly where you want (top/right/etc.) inside the nearest box that has position: relative.

4. Sticky Position

Sticky

This is some scrollable content inside a box. Scroll down and youâ€™ll see the orange box "stick" to the top of this area.

Sticky works only when scrolling happens inside a scrollable container or the page. The element behaves like "relative" at first, and then sticks when it reaches a scroll threshold (like top: 0).

5. Fixed Position

Fixed
Fixed takes the element out of the page flow and sticks it to the screen. This green box stays at the bottom-right corner even when you scroll the entire page.
