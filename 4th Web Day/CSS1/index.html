Three ways to add CSS in HTML file:

  1. Incline CSS
  2. Internal CSS
  3. External CSS
This is done using inline CSS.
Here I have used Internal CSS. Also just for mentioning external CSS can be considered as an extended version of Internal CSS. The Syntax to add external CSS is as follows:
<link rel="stylesheet" href="style.css">
Here I have not used External CSS since this is just a simple webpage showcasing what purpose CSS has and how its used in further learning days I will be adding CSS externally.
